### Docker
#### 功能
* **不穩定所以很多網站不推薦，短時間快速建置可以暫用**
* 使用 Docker 可快速建立、測試和部署應用程式
* Docker 將軟體封裝到名為容器的標準化單位，其中包含程式庫、系統工具、程式碼和執行時間等執行軟體所需的所有項目
* Docker 是容器的作業系統
* 與虛擬機器虛擬化 (免除直接管理的需要) 伺服器硬體的方法相似，容器可虛擬化伺服器的作業系統
